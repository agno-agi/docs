## Code

```python cookbook/14_tools/tavily_tools.py
from agno.agent import Agent
from agno.tools.tavily import TavilyTools

# Example 1: default TavilyTools
agent = Agent(tools=[TavilyTools()], markdown=True)

# Example 1a: TavilyTools with custom API base URL
# useful for self-hosted or alternative Tavily endpoints
agent_custom = Agent(tools=[TavilyTools(api_base_url="https://custom.tavily.com")], markdown=True)
```
