## Quick Start Examples

### Any Model, Same Code

```python
from agno.agent import Agent
from agno.models.openai import OpenAIResponses
from agno.models.anthropic import Claude
from agno.models.google import Gemini

# Switch models with one line
agent = Agent(model=OpenAIResponses(id="gpt-4o"))
agent = Agent(model=Claude(id="claude-sonnet-4-5"))
agent = Agent(model=Gemini(id="gemini-2.0-flash"))

agent.print_response("What is quantum computing?", stream=True)
```

### Knowledge-Powered Agent

```python
from agno.agent import Agent
from agno.knowledge.knowledge import Knowledge
from agno.vectordb.pgvector import PgVector

knowledge = Knowledge(
    vector_db=PgVector(table_name="docs", db_url="postgresql://..."),
)
knowledge.add_content(url="https://example.com/docs.pdf")

agent = Agent(knowledge=knowledge, search_knowledge=True)
agent.print_response("What does the documentation say about authentication?")
```

### Agent with Tools

```python
from agno.agent import Agent
from agno.tools.duckduckgo import DuckDuckGoTools
from agno.tools.yfinance import YFinanceTools

agent = Agent(
    tools=[
        DuckDuckGoTools(),
        YFinanceTools(stock_price=True),
    ],
)
agent.print_response("What's NVDA's stock price and recent news?")
```
