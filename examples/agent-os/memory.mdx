## Overview

Agent memory stores and recalls facts from prior interactions, enabling personalization and long-term context.

<Steps>
  <Step title="Set up a virtual environment">

```shell
python -m venv .venv
source .venv/bin/activate
```

  </Step>
  <Step title="Start Postgres (pgvector) locally">

```shell
docker run --name agno-pg -e POSTGRES_USER=ai -e POSTGRES_PASSWORD=ai -e POSTGRES_DB=ai -p 5532:5432 -d pgvector/pgvector:pg16
```

  </Step>
  <Step title="Install dependencies">

```shell
pip install -U agno fastapi uvicorn sqlalchemy pgvector psycopg openai
```

  </Step>
  <Step title="Set API keys (OpenAI example)">

```shell
export OPENAI_API_KEY=your_key
```

  </Step>
  <Step title="Use the SDK cookbook: cookbook/os/memory_demo.py">

Open `cookbook/os/memory_demo.py` in the Agno SDK:

```python cookbook/os/memory_demo.py
from agno.agent import Agent
from agno.db.postgres.postgres import PostgresDb
from agno.models.openai import OpenAIChat
from agno.os import AgentOS

db_url = "postgresql+psycopg://ai:ai@localhost:5532/ai"

# Setup the database
db = PostgresDb(db_url=db_url)

# Setup the agent with user memories
agent = Agent(
    name="Memory Agent",
    model=OpenAIChat(id="gpt-4o"),
    db=db,
    enable_user_memories=True,
    markdown=True,
)

# Setup the AgentOS App
agent_os = AgentOS(
    description="Example app for memory capabilities",
    os_id="memory-demo",
    agents=[agent],
)
app = agent_os.get_app()

if __name__ == "__main__":
    # Generate a memory
    agent.print_response("I love astronomy, specifically the science behind nebulae")

    # Expose memory APIs
    agent_os.serve(app="memory_demo:app", reload=True)
```

  </Step>
  <Step title="Run and query memories">

```shell
python cookbook/os/memory_demo.py
```

Explore memory endpoints:

```text
http://localhost:8001/memory/{id}/memories
http://localhost:8001/memory/{id}/memories/123
http://localhost:8001/memory/{id}/memories?agent_id=123
http://localhost:8001/memory/{id}/memories?limit=10&page=0&sort_by=created_at&sort_order=desc
```

  </Step>
