Full Anthropic Claude integration with extended thinking support. Claude excels at complex reasoning, code generation, and nuanced analysis.

```python
from agno.agent import Agent
from agno.models.anthropic import Claude

agent = Agent(
    model=Claude(id="claude-sonnet-4-5"),
    markdown=True,
)

agent.print_response("Explain the halting problem", stream=True)
```

### Extended Thinking

Claude's extended thinking shows the model's reasoning process before answering.

```python cookbook/92_models/anthropic/thinking.py
from agno.agent import Agent
from agno.models.anthropic import Claude

agent = Agent(
    model=Claude(
        id="claude-sonnet-4-5",
        max_tokens=16000,
        thinking={"type": "enabled", "budget_tokens": 10000},
    ),
)

agent.print_response(
    "Solve this step by step: A farmer has 17 sheep. All but 9 die. How many are left?",
    stream=True,
    show_full_reasoning=True,
)
```

### Structured Output

```python cookbook/92_models/anthropic/structured_output.py
from pydantic import BaseModel
from agno.agent import Agent
from agno.models.anthropic import Claude

class Analysis(BaseModel):
    summary: str
    key_points: list[str]
    sentiment: str

agent = Agent(
    model=Claude(id="claude-sonnet-4-5"),
    output_schema=Analysis,
)

response = agent.run("Analyze the impact of AI on healthcare")
print(response.content)  # Analysis object
```

### Prompt Caching

Reduce costs and latency for repeated context.

```python cookbook/92_models/anthropic/prompt_caching.py
from agno.agent import Agent
from agno.models.anthropic import Claude

agent = Agent(
    model=Claude(id="claude-sonnet-4-5"),
    instructions=[
        "You are a legal expert. Here is the full text of the contract...",
        # Long context that will be cached
    ],
)

# First call caches the context
agent.print_response("What are the termination clauses?")

# Subsequent calls reuse cached context (faster, cheaper)
agent.print_response("What about liability limitations?")
```

### Financial Analysis with Thinking

```python cookbook/92_models/anthropic/financial_analyst_thinking.py
from agno.agent import Agent
from agno.models.anthropic import Claude
from agno.tools.yfinance import YFinanceTools

agent = Agent(
    model=Claude(
        id="claude-sonnet-4-5",
        thinking={"type": "enabled", "budget_tokens": 8000},
    ),
    tools=[YFinanceTools(stock_price=True, analyst_recommendations=True)],
)

agent.print_response(
    "Analyze NVDA as an investment",
    stream=True,
    show_full_reasoning=True,
)
```
