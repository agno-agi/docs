Enterprise-grade model access through major cloud providers. Same Agent code with your existing cloud infrastructure, compliance, and billing.

```python
from agno.agent import Agent
from agno.models.azure import AzureOpenAI

agent = Agent(
    model=AzureOpenAI(id="gpt-4o"),
    markdown=True,
)

agent.print_response("Summarize our Q4 report", stream=True)
```

## Azure OpenAI (22 examples)

Use OpenAI models through your Azure subscription.

### Basic Chat

```python cookbook/92_models/azure/openai/basic.py
from agno.agent import Agent
from agno.models.azure import AzureOpenAI

agent = Agent(model=AzureOpenAI(id="gpt-4o-mini"), markdown=True)
agent.print_response("What is cloud computing?")
```

### Tool Use

```python cookbook/92_models/azure/openai/tool_use.py
from agno.agent import Agent
from agno.models.azure import AzureOpenAI
from agno.tools.yfinance import YFinanceTools

agent = Agent(
    model=AzureOpenAI(id="gpt-4o"),
    tools=[YFinanceTools(stock_price=True)],
)

agent.print_response("What's Microsoft's stock price?")
```

### Vision

```python cookbook/92_models/azure/ai_foundry/image_agent.py
from agno.agent import Agent
from agno.models.azure import AzureAIFoundry

agent = Agent(model=AzureAIFoundry(id="gpt-4o"))
agent.print_response(
    "Analyze this diagram",
    images=["https://example.com/architecture.png"],
)
```

## AWS Bedrock (22 examples)

Access Claude, Llama, and more through AWS Bedrock.

### Claude on Bedrock


### Bedrock with Tools

```python cookbook/92_models/aws/bedrock/tool_use.py
from agno.agent import Agent
from agno.models.aws import BedrockChat
from agno.tools.duckduckgo import DuckDuckGoTools

agent = Agent(
    model=BedrockChat(id="meta.llama3-70b-instruct-v1:0"),
    tools=[DuckDuckGoTools()],

```

### Vision on Bedrock


## Environment Setup

### Azure OpenAI

```bash
export AZURE_OPENAI_API_KEY=xxx
export AZURE_OPENAI_ENDPOINT=https://your-resource.openai.azure.com/
export AZURE_OPENAI_API_VERSION=2024-02-01
```

### AWS Bedrock

```bash
export AWS_ACCESS_KEY_ID=xxx
export AWS_SECRET_ACCESS_KEY=xxx
export AWS_REGION=us-east-1
```

### Vertex AI

```bash
export GOOGLE_APPLICATION_CREDENTIALS=/path/to/service-account.json
export GOOGLE_CLOUD_PROJECT=your-project-id
```
