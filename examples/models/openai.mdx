Complete OpenAI integration with both Chat Completions and Responses APIs. Supports all GPT models including reasoning models (o1, o3).

```python
from agno.agent import Agent
from agno.models.openai import OpenAIResponses

agent = Agent(
    model=OpenAIResponses(id="gpt-5.2"),
    markdown=True,
)

agent.print_response("Explain quantum entanglement", stream=True)
```

## Models

| Model | Best For | Import |
|-------|----------|--------|
| `gpt-4o` | General tasks, vision, fast | `OpenAIResponses(id="gpt-5.2")` |
| `gpt-4o-mini` | Cost-effective, fast | `OpenAIResponses(id="gpt-4o-mini")` |
| `gpt-5` | Latest capabilities | `OpenAIResponses(id="gpt-5")` |
| `o1` | Complex reasoning | `OpenAIResponses(id="o1")` |
| `o3-mini` | Fast reasoning | `OpenAIResponses(id="o3-mini")` |

### Tool Use


### Audio Input/Output

```python cookbook/92_models/openai/chat/audio_input_agent.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat

agent = Agent(
    model=OpenAIChat(
        id="gpt-4o-audio-preview",
        modalities=["text", "audio"],
        audio={"voice": "alloy", "format": "wav"},
    ),
)

agent.print_response("Hello, how are you?")  # Returns audio
```

### Structured Output

```python cookbook/92_models/openai/responses/structured_output.py
from pydantic import BaseModel
from agno.agent import Agent
from agno.models.openai import OpenAIResponses

class MovieReview(BaseModel):
    title: str
    rating: float
    summary: str

agent = Agent(
    model=OpenAIResponses(id="gpt-5.2"),
    output_schema=MovieReview,
)

response = agent.run("Review the movie Inception")
print(response.content)  # MovieReview object
```
