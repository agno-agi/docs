Full Google Gemini integration with native multimodal support. Gemini handles text, images, video, and audio natively with built-in grounding and search.

```python
from agno.agent import Agent
from agno.models.google import Gemini

agent = Agent(
    model=Gemini(id="gemini-2.0-flash"),
    markdown=True,
)

agent.print_response("Explain the theory of relativity", stream=True)
```

## Models

| Model | Best For | Import |
|-------|----------|--------|
| `gemini-2.0-flash` | Fast, multimodal | `Gemini(id="gemini-2.0-flash")` |

| `gemini-3-flash-preview` | Latest capabilities | `Gemini(id="gemini-3-flash-preview")` |

## Capabilities

### Basic Chat

```python cookbook/92_models/google/gemini/basic.py
from agno.agent import Agent
from agno.models.google import Gemini

agent = Agent(model=Gemini(id="gemini-2.0-flash"), markdown=True)
agent.print_response("Share a 2 sentence horror story")
```

### Thinking Mode

Gemini's thinking mode shows the model's reasoning process.

```python cookbook/92_models/google/gemini/thinking_agent.py
from agno.agent import Agent
from agno.models.google import Gemini

agent = Agent(
    model=Gemini(
        id="gemini-2.5-pro",
        thinking_budget=1280,
        include_thoughts=True,
    ),
    markdown=True,
)

agent.print_response(
    "Three missionaries and three cannibals need to cross a river. "
    "How can all six people get across safely?"
)
```

### Structured Output

```python cookbook/92_models/google/gemini/structured_output.py
from pydantic import BaseModel
from agno.agent import Agent
from agno.models.google import Gemini

class Recipe(BaseModel):
    name: str
    ingredients: list[str]
    instructions: list[str]

agent = Agent(
    model=Gemini(id="gemini-2.0-flash"),
    output_schema=Recipe,
)

response = agent.run("Give me a recipe for pasta carbonara")
print(response.content)  # Recipe object
```
