## Tools Not Working After Load
**Problem**: Tools return None or errors after loading agent

**Solution**: Ensure registry includes all tools used by the agent
```python
registry = Registry(tools=[all_your_tools_here])
agent = get_agent_by_id(db=db, id="agent-id", registry=registry)
```

## Database Connection Errors
**Problem**: Cannot connect to PostgreSQL

**Solution**: Verify database credentials and ensure PostgreSQL is running
```bash
# Test connection
psql -h localhost -U username -d agno_agents
```
