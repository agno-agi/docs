title: CSV URL Reader
---

The **CSV URL Reader** processes CSV files directly from URLs, allowing you to create knowledge bases from remote CSV data sources.

## Code

```python examples/basics/knowledge/concepts/readers/overview/csv_reader_url_async.py
from agno.agent import Agent
from agno.knowledge.knowledge import Knowledge
from agno.vectordb.pgvector import PgVector

db_url = "postgresql+psycopg://ai:ai@localhost:5532/ai"

knowledge = Knowledge(
    # Table name: ai.csv_documents
    vector_db=PgVector(
        table_name="csv_documents",
        db_url=db_url,
    ),
)

# Initialize the Agent with the knowledge
agent = Agent(
    knowledge=knowledge,
    search_knowledge=True,
)

# Comment out after first run
knowledge.insert(
    url="https://agno-public.s3.amazonaws.com/demo_data/IMDB-Movie-Data.csv"
)

# Query the agent
agent.print_response("What genre of movies are present here?", markdown=True)
```
