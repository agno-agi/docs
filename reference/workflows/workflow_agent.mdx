| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `model` | `Model` | Required | Model to use for the agent |
| `instructions` | `str` | Optional | Instructions for the agent. Default instructions are provided. |
| `add_workflow_history` | `bool` | Optional | Whether to add the workflow history to the agent's context |
| `num_history_runs` | `int` | Optional | Number of history runs to include in the agent's context. Default is 5 |
