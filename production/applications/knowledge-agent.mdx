## Overview

An internal knowledge agent that answers questions from company documents, wikis, and knowledge bases using retrieval-augmented generation (RAG).

## Planned Features

- Connect to multiple document sources (Notion, Confluence, Google Drive)
- Semantic search across company knowledge
- Citation of source documents
- Access control integration
- Conversation memory for follow-up questions

## Use Cases

- Employee onboarding assistant
- Policy and procedure lookup
- Technical documentation search
- HR FAQ bot
