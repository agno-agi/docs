An autonomous research agent that investigates topics using Parallel's AI-optimized web search, synthesizes findings from multiple sources, and produces comprehensive research reports with citations.

## What It Does

- Breaks down research questions into sub-queries
- Searches the web using Parallel's AI-optimized search
- Extracts full content from promising sources
- Synthesizes findings with proper citations
- Identifies gaps and suggests follow-up research

## Prerequisites

- Python 3.12+
- Anthropic API key (for Claude)
- Parallel API key (for web search)

## Setup

<Steps>

</Step>
</Steps>

## Agent Configuration

```python
research_agent = Agent(
    name="Research Agent",
    model=Claude(id="claude-sonnet-4-5"),
    system_message=SYSTEM_MESSAGE,
    output_schema=ResearchReport,
    tools=[
        ParallelTools(
            max_results=10,
            enable_search=True,
            enable_extract=True,
        ),
        ReasoningTools(add_instructions=True),
    ],
    markdown=True,
)
```

| Parameter | Purpose |
|-----------|---------|
| `model` | Claude Sonnet for research synthesis |
| `output_schema` | Structured report with findings and sources |
| `ParallelTools` | AI-optimized web search and extraction |
| `max_results` | Maximum sources per search (varies by depth) |
| `enable_extract` | Extract full content from URLs |
| `ReasoningTools` | Plan research approach before searching |

## How It Works

### Research Workflow

```
1. Analyze research question
2. Generate search objective and queries
3. Execute AI-optimized search via Parallel
4. Extract full content from top sources
5. Synthesize findings across sources
6. Identify gaps, iterate if needed
7. Compile final report with citations
```

### Research Depth

| Depth | Sources | Use Case |
|-------|---------|----------|
| `quick` | 3-5 | Fast overview, simple questions |
| `standard` | 5-10 | Balanced research, most questions |
| `comprehensive` | 10-15 | Thorough investigation, complex topics |

### Source Credibility

| Rating | Source Types |
|--------|-------------|
| **High** | Official docs, academic papers, established publications |
| **Medium** | Well-known blogs, industry sites, verified experts |
| **Low** | Personal blogs, forums, unverified sources |

## Troubleshooting

<AccordionGroup>
  <Accordion title="PARALLEL_API_KEY not set">
    Ensure you've exported your Parallel API key:
    ```bash
    export PARALLEL_API_KEY=your-api-key
    ```
    Get a key at [parallel.ai](https://parallel.ai).
  </Accordion>
  <Accordion title="Limited results for niche topics">
    For specialized topics, the agent will report limited sources in the `gaps` field and suggest alternative search approaches.
  </Accordion>
  <Accordion title="Contradictory findings">
    When sources disagree, the agent notes the contradiction, includes confidence level for each claim, and cites sources for both perspectives.
  </Accordion>
</AccordionGroup>
