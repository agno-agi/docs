---
title: "Run Your First AgentOS"
description: "Run a local AgentOS in minutes."
---

Save the following code to `agno_agent.py`:

```python agno_agent.py
from agno.os import AgentOS
from agno.agent import Agent
from agno.tools.mcp import MCPTools
from agno.db.postgres import PostgresDb
from agno.models.anthropic import Claude

agent = Agent(
    model=Claude(id="claude-sonnet-4-5"),
    db=PostgresDb(db_url="postgresql+psycopg://ai:ai@localhost:5532/ai"),
    tools=[MCPTools(url="https://docs.agno.com/mcp")],
    add_history_to_context=True,
    markdown=True,
)

agent_os = AgentOS(agents=[agent], tracing=True)
app = agent_os.get_app()

if __name__ == "__main__":
    agent_os.serve(app="agno_agent:app", reload=True)
```

<Check>

In ~20 lines: an Agent with memory, state, and MCP tools, served as a FastAPI application.

</Check>


## Run your AgentOS

<Steps>
  <Step title="Set up your virtual environment">

    <CodeGroup>
    ```bash Mac
    uv venv --python 3.12
    source .venv/bin/activate
    ```
    ```bash Windows
    uv venv --python 3.12
    .venv\Scripts\activate
    ```
    </CodeGroup>

  </Step>
  <Step title="Install dependencies">
    ```bash
    uv pip install -U agno anthropic mcp 'fastapi[standard]' sqlalchemy
    ```

  </Step>
  <Step title="Export your Anthropic API key">

    <CodeGroup>
    ```bash Mac
    export ANTHROPIC_API_KEY=sk-***
    ```
    ```bash Windows
    setx ANTHROPIC_API_KEY sk-***
    ```
    </CodeGroup>

  </Step>
  <Step title="Run your AgentOS">
    ```bash
    fastapi dev agno_agent.py
    ```

    This starts your AgentOS at `http://localhost:8000`.

  </Step>
</Steps>

Your AgentOS is now running at `http://localhost:8000`.

| Endpoint | What it does |
|----------|--------------|
| `http://localhost:8000` | Connect to the control plane |
| `http://localhost:8000/docs` | Interactive API documentation |
| `http://localhost:8000/config` | View AgentOS configuration |

See [AgentOS API reference](/reference-api/overview) for full documentation.
